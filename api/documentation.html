PDF Tools

/form/fill

Send a Post, get back a file.
  • form= Your pdf file
  • data = json file mapping fields to values

/form/describe

Send a Post, receive JSON data
  • form = Your pdf file

/form/selfreport

Handy tool for identifying field names. Given a PDF form, this will return the same form with official field name filled into each field. Unfortunately, does not work with checkboxes.
  • form = Upload of your PDF form
